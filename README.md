# Turbofan CAD + Blade FEM Study

## Overview
This project presents a conceptual turbofan engine CAD model created in SolidWorks and a simplified FEM study of a blade-like component performed in ANSYS Mechanical.

## Objectives
- develop a conceptual turbofan CAD assembly
- study component layout and interaction
- perform simplified static structural analysis
- perform modal analysis of a blade-like component

## Tools
- SolidWorks
- ANSYS Mechanical

## Scope
### CAD
- conceptual turbofan geometry
- major assemblies: fan, compressor, combustor, turbine, nacelle, exhaust

### FEM
- simplified blade-like component
- static structural analysis
- modal analysis

## Assumptions
- simplified geometry
- simplified boundary conditions
- simplified loading conditions

## Results
- displacement trends
- von Mises stress trends
- natural frequencies
- mode shapes

## Key Takeaways
- geometry strongly affects structural response
- simplified FEM studies are useful for early design understanding
- modal analysis helps identify dynamic behavior of blade-like components
